HTML & CSS Cheat Sheet

HTML Esessentials

  • Document Structure: <!DOCTYPE html>, <html>, <head>, <body>
  • Headings & Paragraphs: <h1> to <h6>, <p>
  • Links & Images: <a href="url">
  • Lists: Ordered <ol>, Unordered <ul>, List Item <li>
  • Tables: <table>, <tr>, <th>, <td>

CSS Basics

  • Selectors: Element, Class, ID
  • Properties & Values: E.G., color: red;, font-size: 16px;
  • Box Model: Margin, Border, Padding, Content
  • Positioning: position: static, relative, absolute, fixed;
  • Flexbox:display: flex;, flex-direction, align-items, justify-content

Advanced Topics

  • Semantic HTML: Use elements according to their intended purpose.
  • Specificity: Understand how to override styles properly.
  • Design: Ensure your site looks good on all devices
  • Accessibility: Use alt tags for images, proper contrast, & semantic HTML for screen readers
